# ピッキング注文表自動生成ツール

就労移行支援事業所でピッキング訓練中の「注文表どうやって作ってるんだ？」という疑問から自作したツールです。
公開用に**完全ダミー化** (商品番号・在庫をランダム生成) しています。

## 主な機能
- 32種類の商品からランダムに6種類を選ぶ
- 選んだ商品に1～3個の数量を割り当て、合計7～9個になるよう調整
- 在庫0になった商品は除外
- Pythonでロジック検証 → VBAでExcel実装
- A3用紙に24枚分まとめて印刷できるフォーマット対応

## 使った技術
- **Python** : ロジック検証・ダミーデータ生成 (random)
- **VBA** : Excel自動化 (Excelマクロ)
- データ匿名化意識 : 実在の商品番号・在庫は一切使用せず、ランダム生成スクリプトで再現

## 使い方
1. dummy_product_generator.py でダミーデータ(product.csv)生成
2. (Python版)main.py で注文表出力
3. (VBA版)生成したCSVをorder_generator.xlsmに読み込み、マクロ実行
